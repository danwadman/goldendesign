Buttons Guidance

Each button should use the standard font size of 1rem (16px) and a height of 42px (which follows the golden ratio).

All buttons should have a focus state border of '1px solid black'

All buttons should have a hover state background 10% darker or lighter than the default state. Darken background colour if using white font. Lighten background colour is using black font
