Guidance

Font

The main font for WebBased content is Google Lato, however appropriate alternatives should be specificed to ensure consistency.

Font size

16px has been chosen as the body font size as this is the recommended standard according to W3C - https://www.w3.org/QA/Tips/font-size

This is reinforced by an in depth article in Smashing Magazine - https://www.smashingmagazine.com/2011/10/16-pixels-body-copy-anything-less-costly-mistake/

There are three font sizes available and they have been chosen based on Fibanaci's golden ratio.

The only exception to this in when Deep Saffron (webbased orange) is used as a background colour for a button. In this screnario, 19px should be used for hte font size and white as the font colour. In order to meet aa accessbility guidelines.

"Two quantities are in the golden ratio if their ratio is the same as the ratio of their sum to the larger of the two quantities. The figure below illustrates the geometric relationship."

Putting this into practice with font sizes, 16px is the body font size, so will be the 'shortest side', then 26px would need to be the 'longest' to keep them in the golden ratio. The sum of these two sizes is 42px.

The line height for all text should be 150%. This is bootstraps standard line height

For more information on the golden ratio, please visit https://en.wikipedia.org/wiki/Golden_ratio

The easiest way to apply these three font sizes is to overide bootstrap default h tag sizes.
